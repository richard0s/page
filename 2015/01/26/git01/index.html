最近一直在研究学习Git，下面就个人对于Git的理解，以及如何使用Git进行总结

Git是什么？

Git是一款分布式版本控制系统

百度百科_Git
维基百科_Git

什么是版本控制系统

版本控制是一种记录若干文件内容变化，以便将来查阅特定版本修订情况的系统。可以参考廖雪峰的官方网站和Git详解之一 Git起步

如何安装Git

For Linux

在控制台输入命令

$ sudo apt-get install git

或者到Git官网下载源码，使用源码进行安装

For Windows

从http://msysgit.github.io/下载msysgit进行安装，使用Git Bash进行Git命令行操作。

配置Git

因为Git是分布式版本控制系统，所以Git每次提交都要知道是谁提交的更改
在命令行输入

$ git config —global user.name “Your Name”

在”Your Name”里填写你的用户名

$ git config —global user.email “email@example.com”

在”email@example.com”里填写你的email地址
